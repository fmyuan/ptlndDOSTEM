Many input and output variables are added. Most of them can be found by searching "CH4".

stefan.h -- define control, raised, or lowered treatement, also with daily climate dataset

Soil_env.h -- define the measured water table

timeconst.h -- define PROJECTION or no PROJECTION; if you turn on PROJECTION, make sure to replace the input files (climate.nc, fire.nc, etc.) defined in the folder "Test_Siter/griddata". See readme.txt in the folder "Test_Siter/griddata"

layerconst.h -- number of layers, MINETHICK[MAX_MIN_LAY]
